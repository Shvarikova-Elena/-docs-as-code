Рисунок должен быть привязан к абзацу, т.е. обтекание текста рисунка должно быть установлено на значение «В тексте». Из этого следует, что версткой рисунка можно управлять через абзац, в котором содержится рисунок. Для оформления такого абзаца используется стиль «РП.Рисунок.Положение».
Данный стиль имеет следующие параметры:
* Отступы:
— Сверху: 0 пт.
—	Снизу: 0 пт.
* Выравнивание: По центру.
* Не отрывать от следующего: Да.
* Стиль следующего абзаца: «_Рисунок».
Граница рисунка должна быть черного цвета с толщиной 0,75 пт.
